# 线上英语记事本 (Online English Notebook) - 产品需求文档 (PRD)

## 1. 项目简介 (Project Introduction)

### 1.1 项目背景
在英语学习过程中，学习者需要随时记录生词、短语、语法点以及学习心得。传统的记事本缺乏针对英语学习的辅助功能（如翻译、发音、复习提醒），而通用的笔记软件又不够垂直。本项目旨在打造一款专为英语学习者设计的在线记事本，结合笔记管理与英语学习工具，提高学习效率。

### 1.2 目标用户
*   英语学生（初高中、大学、备考四六级/考研/雅思托福）
*   职场英语学习者
*   英语爱好者

### 1.3 核心价值
*   **便捷记录**：随时随地记录英语知识点。
*   **学习辅助**：内置翻译、发音等工具，减少切换应用的时间。
*   **科学复习**：基于记忆曲线的复习提醒，帮助巩固记忆。

## 2. 功能需求 (Functional Requirements)

### 2.1 用户模块 (User Module)
| 功能点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| **注册/登录** | 支持邮箱/手机号注册登录，支持第三方（GitHub/Google/微信）登录 | P0 |
| **个人中心** | 用户头像、昵称修改，学习数据统计（累计笔记数、坚持天数） | P1 |
| **多端同步** | 笔记数据云端同步，支持 PC/Mobile 访问 | P0 |

### 2.2 核心笔记模块 (Core Note Module)
| 功能点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| **新建笔记** | 支持富文本编辑（Markdown 支持），支持插入图片、音频 | P0 |
| **笔记分类** | 支持文件夹/标签管理（如：生词、语法、作文、口语） | P0 |
| **笔记搜索** | 全文检索，支持按标签、时间筛选 | P0 |
| **笔记列表** | 展示笔记摘要、创建时间、标签 | P0 |

### 2.3 英语学习特色功能 (English Learning Features)
| 功能点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| **划词翻译** | 在笔记编辑/阅读模式下，选中单词自动弹出释义和音标 | P0 |
| **一键生词本** | 将笔记中的生词一键添加到“生词本”，自动抓取例句 | P1 |
| **每日一句** | 首页展示每日英语金句（支持发音），支持一键保存到笔记 | P2 |
| **复习提醒** | (高级功能) 基于艾宾浩斯遗忘曲线，推送复习提醒 | P2 |
| **语法检查** | (高级功能) 集成 Grammarly 或类似 API，对英文写作进行纠错 | P2 |

## 3. 非功能需求 (Non-functional Requirements)

### 3.1 性能要求
*   页面加载速度 < 2秒。
*   笔记保存实时性：支持自动保存，延迟 < 1秒。

### 3.2 安全要求
*   用户密码加密存储 (如 bcrypt)。
*   HTTPS 加密传输。
*   数据定期备份。

### 3.3 兼容性
*   Web 端：兼容主流浏览器 (Chrome, Edge, Safari, Firefox)。
*   响应式设计，适配手机和平板浏览器。

---

## 4. 技术栈建议 (Tech Stack Recommendation)

### 4.1 前端 (Frontend)
*   **框架**: React 或 Vue.3
*   **UI 组件库**: Ant Design 或 Tailwind CSS
*   **编辑器**: Milkdown 或 Tiptap (支持 Markdown 和所见即所得)

### 4.2 后端 (Backend)
*   **语言**: Node.js (Express/NestJS) 或 Python (FastAPI/Django)
*   **数据库**: MongoDB (适合文档型笔记) 或 PostgreSQL
*   **API**: RESTful 或 GraphQL

### 4.3 部署 (Deployment)
*   **平台**: Vercel (前端) + Render/Railway (后端) 或 云服务器
*   **CI/CD**: GitHub Actions

---

## 5. 项目规划 (Roadmap)

*   **Phase 1 (MVP)**: 完成用户注册登录、基本的笔记增删改查、Markdown 支持。
*   **Phase 2**: 增加标签系统、搜索功能、划词翻译集成。
*   **Phase 3**: 推出生词本功能、复习提醒算法、移动端适配优化。
