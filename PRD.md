# 线上英语移动端 App (Online English Mobile App) - 产品需求文档 (PRD)

## 1. 项目简介 (Project Introduction)

### 1.1 项目背景
现代生活节奏快，英语学习者更倾向于利用碎片化时间（通勤、排队、睡前）进行学习。传统的桌面端学习工具无法满足“随时随地”的需求。本项目旨在打造一款**移动端优先**的英语学习伴侣 App，充分利用手机的便携性、麦克风和摄像头能力，提供沉浸式的学习体验。

### 1.2 目标用户
*   **通勤上班族**：利用碎片时间背单词、听听力。
*   **在校学生**：随时随地查词、拍照搜题、口语跟读。
*   **移动办公人士**：需要快速翻译、商务口语急救。

### 1.3 核心价值
*   **随时随地 (Anytime, Anywhere)**：离线模式支持，无网也能学。
*   **交互丰富 (Rich Interaction)**：语音跟读打分、拍照取词翻译。
*   **碎片化学习 (Micro-learning)**：短小精悍的课程和练习，适配移动场景。

## 2. 功能需求 (Functional Requirements)

### 2.1 移动端核心体验 (Mobile Core Experience)
| 功能点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| **手势操作** | 左滑归档，右滑收藏，双击点赞，符合移动端操作习惯 | P0 |
| **离线模式** | 核心生词本、已下载听力支持离线访问，联网自动同步 | P0 |
| **深色模式** | 适配系统级深色模式，护眼适合夜间学习 | P1 |

### 2.2 学习工具箱 (Learning Toolkit)
| 功能点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| **拍照翻译 (OCR)** | 调用手机摄像头，拍摄英文段落/单词，实时识别并翻译 | P0 |
| **语音口语教练** | 模拟对话场景，用户录音跟读，AI 实时进行发音打分 (调用麦克风) | P0 |
| **锁屏背单词** | (Android特有/iOS Widget) 利用锁屏界面或桌面小组件展示单词 | P1 |
| **听力播放器** | 悬浮窗播放、后台播放控制、倍速调节 (0.5x - 2.0x) | P0 |

### 2.3 记事与复习 (Notes & Review)
| 功能点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| **卡片式笔记** | 适合手机屏幕阅读的卡片式设计，左右滑动切换 | P0 |
| **推送提醒** | 结合艾宾浩斯曲线，发送 Push Notification 提醒复习 | P1 |
| **语音笔记** | 支持快速录制语音笔记，方便记录口语灵感 | P2 |

## 3. 非功能需求 (Non-functional Requirements)

### 3.1 性能与体验
*   **启动速度**: 冷启动 < 1.5秒。
*   **耗电量**: 优化后台播放和定位服务的电量消耗。
*   **安装包大小**: 核心包体控制在 50MB 以内 (Android/iOS)。

### 3.2 兼容性
*   **iOS**: 兼容 iOS 15.0 及以上。
*   **Android**: 兼容 Android 8.0 及以上。
*   **屏幕适配**: 适配刘海屏、挖孔屏以及不同尺寸的平板设备。

---

## 4. 技术栈建议 (Tech Stack Recommendation)

### 4.1 移动客户端 (Mobile Client)
*   **跨平台框架**: Flutter (推荐，性能接近原生) 或 React Native (社区成熟)。
    *   *理由*: 一套代码同时生成 iOS 和 Android 应用，开发效率高。
*   **原生开发 (可选)**: Swift (iOS) / Kotlin (Android) - 如果需要极致性能或深度系统集成。

### 4.2 后端服务 (Backend Service)
*   **API**: RESTful API 或 gRPC (节省流量)。
*   **数据库**: PostgreSQL + Redis (缓存)。
*   **AI 服务**: 接入 Whisper (语音识别) + OpenAI/DeepL (翻译/纠错)。

### 4.3 基础设施
*   **推送服务**: Firebase Cloud Messaging (FCM) / 极光推送。
*   **存储**: AWS S3 / 阿里云 OSS (存储图片、音频)。

---

## 5. 迭代规划 (Roadmap)

*   **v1.0 (MVP)**: 基础账号系统、单词查询、文本笔记、简单的听力播放器。
*   **v1.1**: 引入拍照翻译 (OCR) 和离线缓存功能。
*   **v1.2**: 增加口语跟读评测功能，上线桌面小组件 (Widget)。
*   **v2.0**: 社区打卡功能，排行榜，社交分享。
